# 🎯 Anonymous Aimbot v2.0

A sophisticated aimbot script for Roblox with advanced features and a modern GUI interface.

## ✨ Features

### 🎯 Aimbot System
- **Advanced Targeting**: Intelligent NPC detection with weapon filtering
- **Smooth Aiming**: Customizable smoothing for natural movement
- **FOV Control**: Adjustable field of view for precise targeting
- **Right-Click Activation**: Hold RMB to activate aimbot
- **Visual FOV Circle**: Real-time FOV visualization

### 🔫 Silent Aim
- **Silent Targeting**: Hit targets without visible aim movement
- **Hit Chance Control**: Adjustable success rate (0-100%)
- **Prediction System**: Advanced trajectory calculation
- **Separate FOV**: Independent FOV settings for silent aim

### 👁️ ESP System
- **Head ESP**: Green box highlights on NPC heads
- **Real-time Updates**: Automatic ESP management
- **Performance Optimized**: Efficient rendering system

### 🎨 Visual Enhancements
- **FullBright**: Enhanced visibility in dark areas
- **No Fog**: Remove atmospheric fog for clear vision
- **Particle Effects**: Animated visual effects
- **Modern UI**: Sleek, professional interface

## 🎮 Controls

| Key | Function |
|-----|----------|
| `INSERT` | Toggle GUI |
| `RMB` | Activate Aimbot |
| `F1` | Quick Toggle Aimbot |
| `F2` | Quick Toggle Silent Aim |
| `F3` | Quick Toggle FullBright |
| `F4` | Quick Toggle No Fog |

## 🛠️ Installation

1. **Copy the script** from the repository
2. **Open your Roblox executor** (Synapse X, KRNL, etc.)
3. **Paste the script** into the executor
4. **Execute the script** while in-game
5. **Wait for the startup animation** to complete
6. **Use INSERT key** to toggle the GUI

## ⚙️ Configuration

### Aimbot Settings
- **Smoothing**: 1-20 (Default: 5)
- **FOV**: 10-200 (Default: 60)
- **Scan Cooldown**: 0.15 seconds

### Silent Aim Settings
- **FOV**: 10-300 (Default: 100)
- **Hit Chance**: 0-100% (Default: 100%)
- **Prediction**: Enable/Disable trajectory prediction
